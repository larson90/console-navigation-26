# Console Navigation 26

UX-прототип навигации Cloud.ru (шапка ЛК, меню сервисов, поиск по каталогу).

## Локальный просмотр

1. Установите зависимости (один раз):

```bash
npm install
```

2. Запустите **один** dev-сервер:

```bash
npm run dev
```

3. Откройте в браузере **только этот адрес**:

**http://localhost:5184/**

Порт зафиксирован (`5184`). Если порт занят, Vite сообщит об ошибке — закройте старые процессы (`lsof -i :5184`) и запустите снова.

4. Жёсткое обновление страницы: **Cmd+Shift+R** (Mac) или **Ctrl+Shift+R** (Windows).

### Почему «слетает» хост / не видно изменений

| Причина | Что делать |
|--------|------------|
| Открыт другой порт (`5173`, `5183`…) | Используйте только **5184** |
| Несколько `vite` одновременно | Остановите лишние: `pkill -f "vite --port"` или закройте терминалы |
| Смотрите **деплой** (Vercel/GitLab), а не локально | На проде старая сборка, пока не запушили в `main` |
| Кэш браузера | Cmd+Shift+R |

## Где смотреть фичи

### Иконка маркетплейса (violet + магазин)

1. `http://localhost:5184/`
2. В шапке слева — кнопка с иконкой платформы → выпадающий список
3. Блок **«Другие продукты»** → **«Маркетплейс»**

### Поиск по каталогу

1. Меню **«9 точек»** в шапке → выберите **Прототип 1**, **2** или **3**
2. Поле поиска вверху — фильтрует категории и сервисы в каталоге

Примеры: `Мониторинг`, `Правила алертов`, `Хранение данных`

Категории **не раскрываются автоматически** при вводе — раскройте вручную, чтобы увидеть вложенные сервисы.
